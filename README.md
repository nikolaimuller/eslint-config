Eslint config by mullerstd

## Install

```bash
yarn add -D @mullerstd/eslint-config

// or using npm

npm i @mullerstd/eslint-config --save-dev
```

## Usage

```json
// package.json

{
  "prettier": "@mullerstd/eslint-config/prettier",
  "eslint": {
    "extends": "@mullerstd/eslint-config"
  }
}
```
